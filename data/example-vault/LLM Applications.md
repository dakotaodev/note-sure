# LLM Applications

## Common Patterns
- **[[RAG Systems]]** - Ground responses in external documents
- **Agents** - LLMs making tool calls and decisions
- **Fine-tuning** - Adapt models for specific tasks
- **Prompt Engineering** - Optimize inputs for better outputs

## Popular LLM Models

### Proprietary
- OpenAI GPT-4 / GPT-3.5 Turbo
- Google Claude (via Anthropic)
- Microsoft Copilot

### Open Source
- Meta Llama 2
- Mistral
- Zephyr
- Phi

## Deployment Considerations
- **Latency Requirements** - Real-time vs batch processing
- **Throughput** - How many concurrent requests
- **Cost** - API vs self-hosted
- **Privacy** - Data governance requirements
- **Customization** - Fine-tuning or prompting

## Common Use Cases
1. Chatbots and conversational AI
2. Content generation
3. Code analysis and generation
4. Knowledge extraction
5. Question answering with [[RAG Systems]]

## Development Frameworks
- **LangChain** - Python library for building LLM applications
- **LlamaIndex** - Data framework for LLMs
- **Semantic Kernel** - Microsoft framework
- **Vercel AI** - JavaScript/TypeScript SDK

## Current Focus
Building agentic systems that can leverage personal knowledge bases stored in [[RAG Systems]].
